## LinkedIn post draft

I finished a small browser game project: **S.T.A.L.K.E.R. Clicker**.

It is a vanilla JavaScript clicker game inspired by the atmosphere of survival games set in anomalous zones. The player collects coupons, upgrades click power and protection, levels up artifacts, survives timed emissions, and receives a daily bonus saved through localStorage.

What I focused on:

- modular JavaScript structure without frameworks;
- interactive UI and game state management;
- local save system with `localStorage`;
- audio controls and feedback effects;
- responsive layout for different screen sizes;
- GitHub-ready documentation and project structure.

Tech stack: **HTML, CSS, Vanilla JavaScript**.
